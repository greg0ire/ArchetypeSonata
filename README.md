# Sonata Archetype

## Requirements

- Vagrant 1.8.1
- Ansible 2.0

## Setup

**Clone repository:**

    git clone https://github.com/jordisala1991/ArchetypeSonata.git

    vagrant up

**Add sonata.dev to hosts:**

    echo '192.168.33.99 sonata.dev' | sudo tee --append /etc/hosts

**Open `http://sonata.dev` in your browser.**

**Login to Sonata:**

    user: admin
    password: admin
