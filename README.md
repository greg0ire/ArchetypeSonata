# Sonata Archetype

## Setup

**Clone repository:**

    git clone git@bitbucket.org:runroom/archetype-symfony.git

    vagrant up

**Add sonata.dev to hosts:**

    echo '192.168.33.99 sonata.dev' | sudo tee --append /etc/hosts

**Open `http://sonata.dev` in your browser.**

**Login to Sonata:**

    user: admin
    password: admin
